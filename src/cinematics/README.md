# /src/cinematics

Phase 1 teaser scaffold for CyberTrader: Age of Pantheon.

## Included composition

- `Phase1Teaser`

It currently uses:
- the Eidolon shard core asset
- Void Bloom commodity art
- Blacklight Serum commodity art
- terminal-style copy and gradient lighting

## Run

From repo root:

```bash
npm run generate:cinematic
```

Directly:

```bash
cd src/cinematics
npm install
npm run studio
```

## Render

```bash
cd src/cinematics
npm run still
npm run render
```
