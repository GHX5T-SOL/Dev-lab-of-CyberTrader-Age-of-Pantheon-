## Generation prompt (for SpriteCook or equivalent)

> Minimal cyberpunk Eidolon sigil. Thin-line geometry. Hollow triangle with a single horizontal glitch-interrupt line crossing it. Pure electric cyan `#00F5FF` on deep-void background `#050608`. No gradients, no glow, no lens flare. One stroke, no fills. Feels like a terminal prompt symbol, not a logo.

## Safe area

Keep a margin of `0.5x` the mark's height around all logo placements.

## Don'ts

- ❌ Add a second color to the primary mark
- ❌ Outer glow / drop shadow
- ❌ Tilt or 3D
- ❌ Place on busy imagery without a solid void backplate
